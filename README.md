# CodeAlpha_WebScraping
Python Web Scraping project using Requests, BeautifulSoup and Pandas for CodeAlpha Internship.

# 📚 CodeAlpha - Web Scraping

## Project Overview

This project was completed as part of the **CodeAlpha Python Programming Internship**.

The project demonstrates how to scrape book information from the **Books to Scrape** website using Python.

---

## Features

- Connects to a website
- Scrapes book details
- Extracts:
  - Book Title
  - Price
  - Rating
  - Availability
- Stores the data into a CSV file

---

## Technologies Used

- Python 3
- Requests
- BeautifulSoup4
- Pandas

---

## Website Used

https://books.toscrape.com/

## How to Run

1. Install the required libraries:

```
pip install -r requirements.txt
```

2. Run the program:

```
python web_scraping.py
```
